# Aswin John - Portfolio Website

A professional portfolio website for Aswin John, Mobile & Web Developer | UI/UX Designer, built with Vite.

## About

This portfolio showcases Aswin John's professional experience, technical skills, projects, and education. Built as a modern, responsive single-page application with clean design and smooth animations.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth hover effects and animations
- **Dark/Light Mode**: Automatically adapts to system preferences
- **Single Page Application**: Fast navigation with smooth scrolling
- **Professional Sections**: Experience, Skills, Projects, Education, and Contact

## Files Structure

- `index.html` - Main HTML file with meta tags for SEO
- `src/main.js` - JavaScript entry point with portfolio content
- `src/style.css` - Modern CSS with responsive design
- `portfolio.html` - Standalone version (no server needed)
- `package.json` - Project dependencies

## Running the Website

### Option 1: Standalone HTML (Recommended)
Simply open `portfolio.html` in your web browser to see the complete portfolio.

### Option 2: Development Server
If you have Node.js 20+ installed, you can run:

```bash
npm install
npm run dev
```

Then open your browser to `http://localhost:5173`.

## Portfolio Sections

1. **Hero Section**: Name, title, location, and contact information
2. **About**: Professional summary and background
3. **Experience**: Detailed work history with achievements
4. **Skills**: Technical skills organized by category
5. **Projects**: Highlighted projects with descriptions
6. **Education**: Academic background and certifications
7. **Contact**: Multiple ways to get in touch

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Build Tool**: Vite
- **Styling**: Modern CSS with CSS Grid and Flexbox
- **Design**: Responsive, mobile-first approach

## Customization

To customize this portfolio:
1. Update personal information in `src/main.js`
2. Modify colors and styling in `src/style.css`
3. Add or remove sections as needed
4. Update contact information and links
